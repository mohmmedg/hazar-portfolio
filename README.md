# Hazar Alghanem Portfolio

A modern personal portfolio website built using React, TypeScript, and Vite.

## 🚀 Features
- Responsive design
- Admin dashboard
- CMS-like project management
- Contact form
- Modern UI/UX

## 🛠️ Tech Stack
- React
- TypeScript
- Vite
- Supabase (backend)
- CSS
