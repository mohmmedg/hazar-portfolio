# Hazar Alghanem Portfolio

A modern personal portfolio website built using React, TypeScript, and Vite.

## 🚀 Features
- Responsive design
- Admin dashboard
- CMS-like project management
- Contact form
- Modern UI/UX

## 🛠️ Tech Stack
- React
- TypeScript
- Vite
- Supabase (backend)
- CSS

## 📦 Installation

```bash
npm install
npm run dev
```

## 📁 Environment Variables

Create a `.env` file in the project root and add the following values:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
VITE_CONTACT_WHATSAPP=+963XXXXXXXXX
VITE_CONTACT_EMAIL=example@example.com
VITE_CONTACT_PHONE=+963XXXXXXXXX
VITE_CONTACT_INSTAGRAM=https://instagram.com/yourprofile
```

> Use `.env` for local environment variables. There is no need لملف `.env.local` unless لديك إعدادات خاصة جداً تريد فصلها عن باقي الإعدادات.
